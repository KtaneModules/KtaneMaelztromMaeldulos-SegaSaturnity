On the Subject of Imitation

It’s like a riddle, wrapped in a mystery, inside an enigma…

  • You must use the buttons to adjust each digit on the dial to the proper value before pressing SEND.
  • The buttons below the dial increase or decrease a digit on the dial by one. Their function resets upon a strike.
  • Sum the three digits on the dial. If that number is two digits, sum the digits again. Use this sum in the table below to determine the Cipher Key, along with the amount of buttons that increase the display.
  • Use only the original display value of each digit to determine how much to adjust it up or down. Adjust each digit independently.
  • Adjust each digit that matches the cipher key by twice the amounts given.
  • Decrease each digit 5-9 by the amount equal to the cipher key.
  • Increase each digit 0-4 by the amount equal to the cipher key.
  • If exactly two digits match, increase the left digit by the cipher key if they are even, or decrease the right digit by the cipher key if they are odd.
  • Decrease even digits by the amount of buttons that increase the display.
  • Increase odd digits by the amount of buttons that decrease the display.
  • Before pressing SEND, if any displayed number still matches the cipher key, adjust it up 1 if the cipher key is odd, and down 1 if the cipher key is even.
