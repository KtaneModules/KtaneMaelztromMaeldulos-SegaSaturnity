On the Subject of Imitation

It's like a riddle, wrapped in a mystery, inside an enigma...

  • You must use the buttons to adjust each digit on the dial to the proper value before pressing SEND.
  • The buttons below the dial increase or decrease a digit on the dial by one. Their function resets upon a strike.
  • Sum the three digits on the dial. If the sum is two digits, sum them again. Use this sum in the table below to determine the Cypher Key, along with the amount of buttons that increase the display.
  • Use only the original display value of each digit to determine how much to adjust it up or down. Adjust each digit independently.
  • Adjust each digit that matches the cypher key by double the amounts given.
  • Decrease each digit 5-9 by the amount equal to the cypher key.
  • Increase each digit 0-4 by the amount equal to the cypher key.

  • Increase digits by the amount of buttons that decrease the display.
